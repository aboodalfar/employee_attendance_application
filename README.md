## the app by : php laravel

1- run command line ``` composer install ```
<br/>
2-copy file .env.example  to  .env

and change this lines :

```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=University_Attendance_System
DB_USERNAME=root
DB_PASSWORD=
```
<br/>
3-  run command line ``` php artisan migrate ```
